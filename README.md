An example react frontend with Next.js and typescript that uses mockdata to display (no backend needed).

![CareDash App Screenshot](public/screenshot1.png)

## Getting Started

This project uses pnpm as the package manager.

Clone the repository and cd to the directory

```bash
git clone https://github.com/hvandamm/react-voorbeeld1.git
cd react-voorbeeld1
```

then simply run

```bash
pnpm dev
```
this will install all the required dependencies.

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
